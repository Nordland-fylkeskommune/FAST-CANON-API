# FAST-CANON-API

Fast and simple API for capturing and sending images from a canon camera.

## Getting Started

### Dependencies

* Linux based OS
* libgphoto2
* Python3
* Pip

```
PIP PACKAGES
anyio==3.6.2
click==8.1.3
fastapi==0.87.0
gphoto2==2.3.4
h11==0.14.0
httptools==0.5.0
idna==3.4
pydantic==1.10.2
python-dotenv==0.21.0
PyYAML==6.0
sniffio==1.3.0
starlette==0.21.0
typing_extensions==4.4.0
uvicorn==0.20.0
uvloop==0.17.0
watchfiles==0.18.1
websockets==10.4
```

### Installing

* Clone/fork repo 
```
pip install -r /path/to/requirements.txt
```

### Executing program

```
./start.sh
```
